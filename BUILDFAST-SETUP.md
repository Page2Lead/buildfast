# BuildFast — AI Customer Support System

**Status:** ✅ Complete

---

## **What is BuildFast?**

An ecommerce customer support ticket system with AI auto-resolution:
- 🎫 Support ticket submission form
- 🤖 AI auto-resolution suggestions
- 📊 Live dashboard with tickets table
- ✅ Status management (Open → AI Resolved → Escalated → Closed)
- 📈 Support stats (Total, AI Resolved, Escalated)

---

## **Quick Start**

```bash
cd buildfast
npm install
npm run dev
```

Visit `http://localhost:5173`

---

## **Features**

**Public View:**
- Ecommerce website
- Support contact form
- 6 support categories

**Admin Dashboard:**
- Stats cards (Total tickets, AI resolved, escalated)
- Tickets table with status dropdown
- Ticket detail modal with AI response suggestions
- One-click AI resolution

**Working Logic:**
- Create ticket → appears in dashboard
- Click "View" → see AI suggestion
- Click "Resolve with AI" → ticket auto-resolved
- Change status → instant update
